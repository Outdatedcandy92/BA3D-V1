## Features:

- 250mm^3 Build Volume
- Bedslinger Design
- In-Built Camera for Timelapse and Build Failure Detection
- Filament Tangle and Runout Detection
- Prints up to 0.1mm Layer Resolution
- OctoPrint + Klipper
- Support for minimum 2 Filament AMS System
